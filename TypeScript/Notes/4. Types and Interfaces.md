`TS` Automatically understand all the default `JS` types this is known as **Type Inferencing**

## Type Error

Occurs when a diffferent type of value is other than with the intialized type. 

## Type Annotation

> **annotate**: to add notes or provide explanation

When the type of a value is explicitly explained by suffixing `:data_type` 

> `NOTE`
> 
> In `:data_type` the data_type should be in **lower_case**
