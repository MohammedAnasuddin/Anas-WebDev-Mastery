## Type Assertions

> what's as, of, in  keywords

Assures the typescript that value is of the type mentioned for sure.

> `NOTE`
> 
> It does not convert the type. 

> **Handling `type` of Error in Catch**
> 
> Add a type guard on the received error 
> 
> ```ts
> catch(error){
>     if(error instanceof Error){
>     //Handle the error
>     }
> }
> 
> 
> ```
