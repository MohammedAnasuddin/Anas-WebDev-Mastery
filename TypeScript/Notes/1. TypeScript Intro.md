# Intro to TypeScript

## The Problem

1. **Freedom** : In `JS`, values can be of any types which makes developers Lazy and leads to bad code.

2. **Loose Docs**: Effincient Docs cannot be crafted (by tools) since Data Types are not specified.

> `NOTE`
> 
> TypeScript is an add-on to the `JS` *it cannot run on it's own.*

The TypeScript add-on gets the features as:

- **Type Checker**:  Enusres the values are of same data type as intented.
