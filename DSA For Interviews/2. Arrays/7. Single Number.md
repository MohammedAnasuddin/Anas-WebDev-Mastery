# 136.  Missing Number

> Given a **non-empty** array of integers `nums`, every element appears *twice* except for one. Find that single one.
> 
> You must implement a solution with a linear runtime complexity and use only constant extra space.

## Approach

`XOR` all the element swith each other

> `XOR(^)` will return 0 if operated on same element.
> 
> 
> 
> > 0 ^ 0 = 0 (same -> 0)
> > 0 ^ 1 = 1 (different -> 1)
> 
> Using this if all the elements are XOR then the value wibb be zero 
> then finally this ) and the single number is XOR , it results in Single Number return it.

```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var singleNumber = function(nums) {
    let xor =0;

    for(let i=0;i<nums.length;i++){
        xor = xor^nums[i];
    }

    return xor;
};
```
