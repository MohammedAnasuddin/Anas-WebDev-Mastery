# 283. Move Zereos

> Given an integer array `nums`, move all `0`'s to the end of it while maintaining the relative order of the non-zero elements.
> 
> **Note** that you must do this in-place without making a copy of the array.
> 
> **Example 1:**
> 
> **Input:** nums = [0,1,0,3,12]
> **Output:** [1,3,12,0,0]

## Approach

Iterate throught the array and also maintain a pointer-x which represents where teh next number should filled.

After all numbers are filled then iterate from x to fill all the zeros

```js
/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
var moveZeroes = function(nums) {
    let i,x=0;

    for(i=0;i<nums.length;i++){
        if(nums[i]!=0){
            nums[x]= nums[i];
            x++;
        }
    }

    for(let z=x; z<nums.length;z++){
        nums[z] =0;
    }
};
```
